# Astrul: S³C Student Portal
Astrul is a modern student support portal for NUST. It brings academic data and support services into one fast and simple interface.

## Core Features
- **Centralized Dashboard**: See your Qalam grades, LMS deadlines, and class schedule in one place.
- **Fast Support**: Submit requests for any campus issue in just 4 taps.
- **Real-time Chat**: Talk directly to staff members about your requests.
- **Smart Tracking**: See exactly where your request is in the pipeline with our color-coded SLA rings.
- **Persistent Data**: No data is ever lost on reload. Your chats and requests are saved automatically.
- **Mobile Optimized**: A premium experience for both students and staff on any device.

## Getting Started
To run the project locally, follow these steps:

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start the development server**:
    ```bash
    npm run dev
    ```

3.  **View the app**:
    Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack
- **Frontend**: Next.js 15, React, Tailwind CSS.
- **State Management**: Zustand with persistent storage.
- **Icons**: Lucide React.
- **Fonts**: Geist Sans and Geist Mono.
